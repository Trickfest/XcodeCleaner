# App Icon Assets

`XcodeInspiredBroomSweep/` contains the current `XcodeCleaner` icon draft: an Xcode-like blueprint tile with a centered hand broom and a subtle dust sweep.

To regenerate the PNG preview and Xcode-friendly `AppIcon.appiconset`, run:

```bash
swift Assets/AppIcon/generate_hammer_broom_icon.swift
```
